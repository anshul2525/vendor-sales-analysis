# Vendor Sales Analysis

Dataset - 

## Overview
This project is an end-to-end vendor sales and inventory analysis built using
SQL, Python (Jupyter Notebook), and Power BI.

The objective is to evaluate vendor performance, profitability, and inventory
efficiency by creating a consolidated vendor sales summary and deriving insights
through analysis and dashboards.

## Tech Stack
- SQL (CTEs, joins, aggregations)
- Python (Pandas, SQLAlchemy, SQLite)
- Jupyter Notebook
- Power BI

---

## Project Flow
1. Raw inventory and sales data is ingested into a SQLite database using Python scripts.
2. Vendor-level sales summary is generated using SQL logic and stored in the database.
3. Feature engineering and data cleaning are applied to derive business metrics.
4. Exploratory Data Analysis (EDA) is performed using Jupyter Notebooks.
5. Final insights are visualized using an interactive Power BI dashboard.
